Project "Text Ninja" Overview

Text Ninja is a strategic enhancement designed to complement the custom conversational AI agent, Gemini, addressing two significant challenges: cognitive overload and overgeneralization in responses. As part of the Snappy project, Text Ninja aims to improve the interaction quality when Gemini operates in active chatbot mode during conversations.

  1. Cognitive Overload: During active engagement, the Gemini conversational agent often provides responses that, while correct, contain excessive information. This can overwhelm the user, leading to a cognitive overload that hinders effective communication.
  2. Overgeneralization: When the chatbot takes control of the conversation, it may not always have sufficient context to provide precise answers. As a result, the responses can be overly general, failing to meet the specific informational needs of the participants.

To tackle these issues, Text Ninja incorporates two main modules: Text Summarization and Text Contextualization.

  1. Text Summarization: This module employs an extractive approach utilizing a multilingual pre-trained BERT neural network model. It effectively reduces the length of chatbot responses while preserving the semantic value of the original output from the Gemini model.
  2. Text Contextualization: Inspired by NLP text augmentation techniques, this module performs contextualized keyword data augmentation. It generates new text based on a base text (such as the response from the Gemini model or the output of the summarization module) and a set of relevant keywords that represent the conversation context or intent. This is achieved using a transformer-based neural network with an AutoSeqtoSeq architecture and a multilingual pre-trained BERT model for Next Word Prediction tasks.

Following are some keypoints that has been focused on:

  • Dual Module Design : Incorporates both text summarization and contextualization to enhance text processing capabilities.
  • Advanced NLP Techniques : Utilizes state-of-the-art models like BERT for effective extraction and contextualization of text.
  • Semantic Preservation : Ensures that the generated text maintains the original meaning and context, improving user interaction in conversational AI.
